# Мой первый проект на GitHub

Этот репозиторий создан в рамках лабораторной работы по изучению возможностей GitHub через веб-интерфейс без использования командной строки.

## Цель проекта
Освоить базовые навыки работы с GitHub:
- создание и настройка репозиториев;
- редактирование файлов онлайн;
- работа с Issues;
- создание веток и Pull Request;
- публикация сайта с помощью GitHub Pages.
